# music

> A Vue.js project

#使用技术
vue-router vuex es6 webpack 

#页面
推荐页面（上swiper下列表，点击列表，跳转到详情页）
歌手页面（点击歌手列表跳转到歌手详情页）
排行榜页面（上tab切换下排行列表，点击列表跳详情页）
播放器页面（中播放器下播放设置，暂停、上下曲、喜欢等）
